# Travel With Sarfraz

A modern and visually appealing travel landing page built using HTML5 and CSS3. The website showcases beautiful travel destinations with immersive parallax scrolling effects, fixed navigation, and engaging travel-themed content.

## Features

- Fixed Navigation Bar
- Full-Screen Hero Section
- Parallax Scrolling Backgrounds
- Mountain, Beach, Forest, City & Village Sections
- Smooth Hover Effects
- Modern Typography with Google Fonts
- Responsive Layout Structure
- Clean and Organized Code

## Technologies Used

- HTML5
- CSS3
- Google Fonts
